## Linguagens de programação

### C++
- multiparadima: foco em OOP
- suporta genéricos
- utilizado em sistemas financeiros de alta performance

### C
- paradigma procedural
- opencl implementado em C
- ligação de SGBDs com código é feito em C

### G-Code
- Especifica instruções para máquinas CNC realizarem.
- Semelhante a uma lista de comandos executados em tempo real
- Paradigma imperativo e procedural

### SQL
- paradigma declarativo
- tradução interpretada

### Javascript


### Java
- paradigma OOP e funcional (funções lambda)
- tradução compilada e interpretada

### HTML
- criado em 1991
- paradigma declarativo
- composto por: tags, elementos e atributos

### Lua
- usada para extensão de sistemas: plugins e scripts para programas e jogos
- traduçao: interpretada
- linguagem de alto nível
- tipagem variável
- multiparadigma: funcional, OOP

### Go Lang
- tradução compilada
- multiparadigma: concorrentes (goroutines e channels) e parcialmente OOP (structs e interfaces)

### Scratch
- programação em blocos


### COBOL
- atualmente tem suporte a OOP
- paradigma procedural
