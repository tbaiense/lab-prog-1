# Linguagem de programação
- Possui regras de sintaxe e semântica, código-fonte.

**Sintaxe**: Forma e formato de escrita, regras de como especificar comandos na linguagem.  
**Semântica**: Significado lógico dos comandos.  
**Código-fonte**: Especificação legível dos comandos em linguagem de programação.  

## Paradigma
Estilo de escrever e usar a linguagem de programação.

Paradigmas:
1. Imperativa
2. Orientado a objeto
3. Funcional: permite o uso de closures (funções como objetos de primeira classe)
4. Declarativo

> Uma linguagem podem ser **multiparadigma**.
