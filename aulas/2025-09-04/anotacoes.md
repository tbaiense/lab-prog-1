# Linguagem de programação Python

Características:
- Modular
- Multiplataforma
- Suporta POO
- Tipagem forte

Criando variáveis:
- Constante: `const MINHA_CONST`
- String (não possui char): `"` o `'`
- Float
- List
- Int
- Boolean: `True`

Operadores:
- `/`
- `//`
- `%`
- `**`
- `++`

Ordem de prescedência:
1. `()`
2. `**`
3. `/` `%` `*`

## Strings
- formatadas: `f"uma string formatada com { idade }"` 
- Não permite concatenação com outros tipos

## Funções

- `print()` ou `print(str)` ou `print(str, varToStore)`
- `int(str)`: conversão

## Condicionais

```
if 10 > 0: 
    print("ola mundo")
else: 
    print("Não é maior que zero")
```

## Comentários

- uma linha: `# este é um comentário`

# Criação de variáveis

- Dinâmica de armazenamento
- Dinâmica de transmissão para a CPU
