# Executando

Para executar o programa utilize o Python na versão 3x, indicando o script
"view_biblioteca.py".

Exemplo:

```
python3 view_biblioteca.py
```

Em SOs *nix também é possível executar diretamente como um script, desde que
o arquivo possua o modo de execução habilitado:

```
./view_biblioteca.py
```
